# User Guide - Church Guest Registration System

This guide is for church staff who will be using the guest registration system to manage visitors and approve registrations.

## 🎯 Getting Started

### Creating Your Account

1. **Get Registration Code** - Ask your church administrator for either:
   - **Admin Code** - Full access to all features
   - **General Code** - Pre-approver access only

2. **Sign Up Process:**
   - Go to the admin login page
   - Click "Create Account"
   - Fill in your email, phone, and password
   - Enter your registration code
   - Click "Create Account"

3. **Sign In:**
   - Use your email and password to log in
   - You'll be taken to the admin dashboard

### Understanding User Roles

**👑 Admin Users Can:**
- View and manage all guest registrations
- Approve or deny guests at any stage
- Manage user accounts and assign roles
- Update their own profile information
- Access all system features

**🔍 Pre-Approver Users Can:**
- View guests pending pre-approval
- Pre-approve or deny initial submissions
- Receive SMS/email notifications for new guests

## 📱 Navigating the System

### Admin Dashboard

The main dashboard shows all guest registrations with:
- **Guest photo and basic info** (name, email, visit date)
- **Status indicators** with color coding:
  - 🟡 **Pending Pre-Approval** - Needs initial review
  - 🔵 **Pending** - Pre-approved, needs final approval
  - 🟢 **Approved** - Fully approved with QR code
  - 🔴 **Denied** - Registration rejected

### Mobile Menu

On mobile devices, use the hamburger menu (☰) to access:
- **Dashboard** - Main guest list
- **Manual Verification** - Verify guest passes
- **User Management** - Manage roles (Admin only)
- **Profile** - Update your information
- **Sign Out** - Log out of the system

## 👥 Managing Guest Registrations

### Viewing Guest Details

1. **Click on any guest** in the list to open their details
2. **Review information:**
   - Personal details (name, email, phone)
   - Visit information (date, time, number of guests)
   - Vehicle information
   - Special needs or allergies
   - Children information (if applicable)
   - Photos (guest and children)

### Pre-Approval Process (Pre-Approvers)

When a new guest registers:

1. **You'll receive a notification** via SMS/email
2. **Review the guest details** in the dashboard
3. **Make a decision:**
   - **Pre-Approve** - Guest looks good, send to final approval
   - **Deny** - Reject the registration with a reason

### Final Approval Process (Admins)

For pre-approved guests:

1. **Review pre-approved guests** (blue status)
2. **Generate pass credentials:**
   - **Approve** - Creates QR code and access credentials
   - **Deny** - Reject with reason

### Guest Status Meanings

- **Pending Pre-Approval** 🟡 - Just submitted, needs initial review
- **Pending** 🔵 - Pre-approved, waiting for final approval
- **Approved** 🟢 - Fully approved, has QR code and access
- **Pre-Approval Denied** 🔴 - Rejected at pre-approval stage
- **Denied** 🔴 - Rejected at final approval stage

## 🔐 Manual Verification

Use this feature to verify guest passes on-site:

1. **Go to Manual Verification** (from menu or dashboard button)
2. **Enter the pass ID** from the guest's QR code or email
3. **System shows:**
   - Guest photo and information
   - Pass validity status
   - Visit details

## 👤 User Management (Admin Only)

### Managing User Roles

1. **Go to User Management** from the menu
2. **View all system users** with their current roles
3. **Assign/Remove Roles:**
   - Check/uncheck **Admin** role
   - Check/uncheck **Pre-Approver** role
   - Changes save automatically

### Role Combinations

- **Admin Only** - Full system access, no pre-approval notifications
- **Pre-Approver Only** - Can pre-approve, gets notifications
- **Admin + Pre-Approver** - Full access plus notifications
- **No Roles** - Cannot access admin features

## ⚙️ Profile Management

### Updating Your Information

1. **Go to Profile** from the menu
2. **Update available fields** (depending on system configuration):
   - Email address (if email confirmation is disabled)
   - Phone number (if phone confirmation is disabled)
3. **Save changes**

*Note: Some fields may be grayed out based on your church's security settings.*

## 📧 Notifications

### SMS Notifications

Pre-approvers receive SMS messages when:
- New guests submit registrations
- Messages include guest name, visit date, and approval link

### Email Notifications

Similar notifications are sent via email with:
- Guest details
- Direct links to the admin dashboard
- Action buttons for quick approval/denial

## 📱 Mobile Tips

- **Guest Details** - All information stacks vertically for easy reading
- **Menu** - Tap the hamburger icon (☰) to access all features
- **Photos** - Tap guest photos to see them larger
- **Forms** - All forms are optimized for mobile typing

- **Use landscape mode** for easier reading on small screens
- **Zoom in** on guest photos if needed to verify identity
- **Keep the app bookmarked** for quick access
- **Log out** when finished for security

## 🆘 Common Tasks

### How to Pre-Approve a Guest
1. Open guest details from dashboard
2. Review all information and photos
3. Click "Pre-Approve" button
4. Guest moves to "Pending" status

### How to Generate Final Approval
1. Find guest with "Pending" (blue) status
2. Open guest details
3. Click "Approve and Generate Pass"
4. Guest receives QR code and access information

### How to Deny a Registration
1. Open guest details
2. Click "Deny" button
3. Optionally add a reason
4. Guest is notified of denial

### How to Check a Guest Pass
1. Go to "Manual Verification"
2. Enter the pass ID from guest's QR code
3. Verify guest photo matches person
4. Check visit date and details

### How to Add New Admin Users
1. Give them the admin registration code
2. Have them create account at login page
3. Go to User Management
4. Assign appropriate roles

## 🔒 Security Best Practices

- **Log out** when finished using the system
- **Don't share** registration codes
- **Verify guest photos** match the person checking in
- **Keep your password secure** and don't share accounts
- **Report suspicious** registrations to church leadership

## ❓ Troubleshooting

### Can't Log In
- Check email and password spelling
- Ensure you have an account (contact admin for registration code)
- Try refreshing the page

### Not Receiving Notifications
- Check your phone number is correct in your profile
- Verify SMS/email settings with church administrator
- Check spam folder for email notifications

### Guest Photos Not Loading
- Check your internet connection
- Try refreshing the page
- Photos may take a moment to load on slower connections

### Mobile Menu Not Working
- Try refreshing the page
- Ensure you're using a modern browser
- Clear browser cache if issues persist
